Loops and Looping

Loops allow you to iterate code by telling the computer/browser how many times it should run a certain block of code.

Loops are useful because they let you save time and allow you to write significantly less code.

There are three major types of loops

  • For Loop
  • While Loop
  • Do While Loop

Each of the above loops accomplish essentially the same thing; improving code efficiently and preventing redundancy.

While Loops

While loops state that while something is true, the loop will continue to run until that something is no longer true. Typically a while loop is used when you do not have a variable to set up to initialize.

while [condition] { [do something while the above is true]; [iterator]; }

While loops are terminated or ended by using some kind of iterator inside of its code block.

Do While Loop

Do While Loops will always run at least once. They are a variable of the while loop, except they always execute first then evaluate or consider their condition.
