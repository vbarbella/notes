Global and Local Variables

Global and local variables or scope defines where in our code an element exists. A global element tends to exist outside of a function or object. Whereas a local element exists inside a function or object. Local elements do not run automatically with the program. they have to be invoked or specifically used first.

In global scope, variables can be used by anything in our code, including all functions and objects.

In local scope, variables can only be used by the function or objects that it's declared within.
